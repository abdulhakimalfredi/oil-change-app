# Oil Change App (سبيد أويل)

تطبيق حجز مواعيد تغيير الزيت مع أتمتة n8n و AI.

## 🛠️ التقنيات

| Frontend | Backend | Automation |
|----------|---------|------------|
| React | Laravel | n8n |
| Axios | Sanctum | Google Gemini AI |
| CSS | MySQL | Twilio (WhatsApp/SMS) |

## ✨ المميزات

- حجز مواعيد تغيير الزيت
- تأكيد المواعيد تلقائياً عبر WhatsApp/SMS
- رسائل مولّدة بالذكاء الاصطناعي بالعربي
