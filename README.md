# 🛢️ Oil Change App (سبيد أويل) - Automation Lab

تطبيق تجريبي مخصص لاستكشاف وتطبيق مفاهيم الأتمتة (Automation) وربط الأنظمة باستخدام **n8n**.

---

## 🤖 حول هذا المشروع
هذا المشروع ليس الهدف منه بناء نظام برمجي من الصفر، بل هو **بيئة تجريبية (Sandbox)** تم إنشاؤها لغرض:
* تجربة أتمتة تدفق البيانات بين تطبيقات الويب وأدوات الطرف الثالث.
* إعداد سير عمل (Workflow) متكامل باستخدام **n8n**.

> **ملاحظة تقنية:** تم توليد كود الـ Frontend (React) والـ Backend (Laravel) باستخدام **الذكاء الاصطناعي (AI)** كنموذج أولي سريع (Boilerplate)، وذلك لاختصار وقت بناء الواجهات والتركيز الكامل على هندسة الأتمتة والربط.

---

## ⚙️ سيناريو الأتمتة (The n8n Workflow)
الهدف الأساسي هو أتمتة رحلة العميل بعد حجز الموعد:
1. **الطلب:** يتم إرسال بيانات الحجز من واجهة المستخدم إلى الـ Backend API.
2. **الـ Webhook:** يقوم Laravel بإرسال تنبيه فوراً إلى **n8n**.
3. **الذكاء الاصطناعي:** يقوم n8n بتمرير البيانات إلى **Google Gemini AI** لصياغة رسالة تأكيد احترافية باللغة العربية.
4. **التنبيه:** يتم إرسال الرسالة النهائية للعميل عبر **WhatsApp** باستخدام **Twilio**.

---

## 🛠️ الأدوات المستخدمة في التجربة
| المجال | التقنية |
| :--- | :--- |
| **محرك الأتمتة** | **n8n** |
| **الذكاء الاصطناعي** | **Google Gemini AI** |
| **الخلفية (API)** | **Laravel** |
| **الواجهة الأمامية** | **React** |
| **قاعدة البيانات** | **MySQL** |
| **التواصل** | **Twilio** (WhatsApp/SMS) |

---

## 📂 محتويات المستودع
* كود الربط والأتمتة.
* ملف الـ JSON الخاص بسير عمل n8n (موجود في مجلد `/automation`).
* الواجهات البرمجية (API Endpoints) المخصصة لاستقبال وإرسال الـ Webhooks.
